An experimental auto-player bot that uses DOM-based “vision,” decision logic, and adaptive rule learning to automate gameplay in browser environments.

# 🤖 Auto Player Bot

A simple AI bot that plays browser games using DOM-based "vision" and rule-based learning.

## Features
- Reads game state from screen (DOM)
- Makes decisions (jump / wait)
- Learns patterns over time
- Inject into browser games

## How to Use
1. Open a browser game
2. Inject the script (DevTools or bookmarklet)
3. Click "Start Bot"
